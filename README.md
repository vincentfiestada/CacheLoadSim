CacheLoadSim
============

Simulator for load operation between Main Memory and Cache

Supports FIFO, LRU, and LFU replacement algorithms
Supports Direct, Set Associative and Fully Associative Mapping algorithm

This project is built for Windows 8.1
I haven't submitted it to the Windows Store so you have to download and build from source using [Visual Studio 2013](http://visualstudio.com/)
